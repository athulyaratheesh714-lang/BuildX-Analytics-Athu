# IPL Analytics Dashboard

## Dataset
IPL Dataset (2008–2025)

## Why I Selected This Dataset
I selected the IPL dataset because cricket analytics provides valuable insights into team and player performance over multiple seasons. It also offers opportunities to practice data cleaning, SQL analysis, and dashboard creation.

## Business Questions Answered
1. Which team has scored the highest total runs?
2. Who are the top run scorers in IPL history?
3. How has scoring performance changed across seasons?

## Tools Used
- Python (Pandas, NumPy)
- SQLite
- Power BI
- GitHub

## Key Insight
Mumbai Indians and Chennai Super Kings consistently rank among the highest-performing teams in terms of total runs, while batting performance has generally improved over the years.
